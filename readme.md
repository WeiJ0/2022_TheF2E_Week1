## 使用套件

- `swiper` - 輪播工具用在時程的移動端橫向移動
- `gsap` - 滾動視差及所有動態效果
- `lottie-files` - 讀取 AE 輸出 Json 轉換成動畫

## 用到的新玩意

- CSS - `:is` 語法，可以快速選定多個群組
